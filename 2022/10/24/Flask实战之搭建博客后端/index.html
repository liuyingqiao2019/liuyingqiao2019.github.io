API认证

为了简化使用 token 认证时客户端和服务器之间的交互，可以使用 Flask-HTTPAuth 插件

当客户端想要开始与 API 交互时，它需要使用用户名和密码进行 Basic Auth 验证，然后获得一个临时 token。只要 token 有效，客户端就可以发送附带 token 的 API 请求以通过认证。一旦 token 到期，需要申请新的 token。
